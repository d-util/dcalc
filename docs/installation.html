Installation

To install Dcalc, follow these steps:

WINDOWS (NO REQUIREMENTS):
  1. Download the latest version from the releases page.
  2. Scroll down to the Assets tab, and click on dcalc.exe.
  3. Run the installer and follow the on-screen instructions.
  4. Due to the exe being new to your computer, You will be asked if you really need this package. Just click Keep.
  5. Click RUN EXE.
  6. You should see a prompt like this:
    Type the expression at the prompt.
    >>>_
CROSS-PLATFORM (REQUIREMENTS: PYTHON 3.9 OR LATER)
  1. Make sure that you have Python 3.9 or later, by typing python --version.
    If it says, 'python' is not recognized as an internal or external command, operable program or batch file., or if you dont have Python 3, Go to the Python Download Page and Download the latest version. (IMPORTANT: DCALC CURRENTLY SUPPORTS 3.9 - 3.12.7. NEWER VERSIONS ARE STILL BEING TESTED. IF YOU SEE BUGS WITH NEWER VERSIONS, GO TO THE BUG REPORT PAGE [lol it is coming soon XD] )
  2. Download the latest version from the releases page.
  3. Then now type python --version and make sure you get something like Python 3.12.7
  4. You're ready to scroll down to the assets page, and click on the zip.
  5. more instructions at the releases page.
